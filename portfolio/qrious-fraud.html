Behavioral Fraud Detection

Behavior-based risk scoring and anomaly detection for digital fraud screening.

Classification Imbalanced data XGBoost Temporal Feature Extraction Behavioral Modeling

Overview

In collaboration with Qrious Insight, we built a scalable fraud detection framework grounded entirely in behavioral data. I led the exploratory analysis of 30-day pre-ban web activity, identifying statistically significant differences in category usage and temporal patterns between fraud and clean users.

Based on these insights, I engineered features across web activity, app activity, and search behavior, and benchmarked multiple model families. Our final XGBoost-based screening model generates monthly fraud confidence scores, reducing manual review burden while improving data quality for downstream analytics.

Key outcome: The model is expected to reduce incentive costs by 10% annually.
